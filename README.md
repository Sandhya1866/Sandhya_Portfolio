# Portfolio Website - Sandhyarani Rajulapati

A modern, responsive portfolio website showcasing skills, projects, and education.

## Features

- Responsive design that works on all devices
- Smooth scrolling navigation
- Animated sections with scroll effects
- Mobile-friendly hamburger menu
- Professional layout with modern UI/UX

## How to Run Your Portfolio

### Method 1: Double-Click (Easiest)
1. **Double-click `RUN_PORTFOLIO.bat`** - This will automatically open your portfolio in your default browser

### Method 2: Manual
1. **Right-click on `index.html`**
2. Select "Open with" → Choose your browser (Chrome, Firefox, Edge, etc.)

### Method 3: Drag and Drop
1. **Drag `index.html`** into any open browser window

## Your Image is Already Added! ✓

Your profile image "Sandhya .jpg" is already configured and will display automatically when you open the portfolio.

### To Change Your Image (Optional):
- Replace "Sandhya .jpg" with a new image
- Keep the same filename OR update line 47 in `index.html`
- Recommended: Square format (350x350px or larger) for best results

## Customize Your Portfolio

- **Change Colors**: Edit CSS variables at the top of `styles.css`
- **Update Content**: Modify text in `index.html`
- **Add Projects**: Copy and paste project card sections
